How to Install Admin Templates
1. Ensure Node.js is Installed

Make sure you have Node.js installed on your machine. We recommend version 20.11.0.

If not already installed, download and install Node.js from the official website.

2. Install Node Modules

Open the command prompt and navigate to your project directory.

To install the required Node modules, run the following command:

npm install

3. For Production-Level Start

For a production-level start, use the following command:

npm run dev

4. Start the Application

To start the application, run the following command:

npm run start
